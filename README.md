# Quadratic Equation Solver (Python)

A simple Python program that solves quadratic equations of the form:

ax² + bx + c = 0

The user enters the coefficients **a**, **b**, and **c**, and the program calculates the discriminant and the roots using the quadratic formula.

## Features

- Calculates the discriminant (b² - 4ac)
- Determines if the equation has real solutions
- Computes both roots if the discriminant is positive
- Verifies the solutions by substituting them back into the equation

## How It Works

The program uses the quadratic formula:

x = (-b ± √(b² - 4ac)) / (2a)

Steps:
1. The user enters the coefficients.
2. The program calculates the discriminant.
3. If the discriminant is negative, the program exits.
4. Otherwise, it calculates the two roots.
